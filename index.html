Marche à suivre

git init : Initialiser git dans un nouveau dossier

git log + nom du fichier : Permet de voir les changements dans un fichier

git diff : Permet de voir la différence entre 2 fichiers

Lorsque l'on veut push un changement par exemple dans index.html, 3 étapes : git add + nom du fichier git commit -m "Commentaire" git push

Si l'on veut supprimer un fichier : git rm index.html git commit -m "Suppression" git push

Lors de la création du repos, laisser le profil en "Public" et la license en "None"
Mettre le même nom pour le repos ET le dossier local ( Pour ne pas s'embrouiller )
